# She Can NGO Landing Page

A modern and responsive NGO website created for **She Can Foundation** as part of a Web Development Internship Task.

## 🌸 Project Overview

This website is designed to spread awareness about women empowerment, education, community support, and volunteering opportunities. The design focuses on a clean UI, smooth user experience, and responsive layout.

## ✨ Features

- Responsive Design
- Modern NGO Landing Page
- Hero Section
- About Section
- Mission Cards
- Volunteer / Join Us Section
- Testimonials
- Image Gallery
- Contact Section
- Animated UI Elements
- Smooth Scrolling
- Hover Effects

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎨 Theme

The website uses a soft and modern color palette inspired by women empowerment themes:
- Pink
- Purple
- White gradients

## 📱 Responsive Design

The website is fully responsive and works smoothly on:
- Desktop
- Tablet
- Mobile Devices
